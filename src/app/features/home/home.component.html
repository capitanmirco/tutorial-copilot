Come iniziare con Copilot

  1. Installa l'estensione

    Installa l'estensione GitHub Copilot per VS Code dal marketplace.

  2. Configura l'accesso

    Accedi con il tuo account GitHub e configura l'accesso.

  3. Inizia a scrivere codice

    Copilot inizierà automaticamente a suggerire codice mentre scrivi.
