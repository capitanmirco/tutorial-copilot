Come Utilizzare GitHub Copilot

1. Inserisci un commento descrittivo

Descrivi cosa vuoi ottenere con un commento chiaro e dettagliato.

// Crea una funzione che calcola la media di un array di numeri

2. Lascia che Copilot generi il codice

Premi Tab o accetta il suggerimento quando appare.

// Crea una funzione che calcola la media di un array di numeri
function calculateAverage(numbers) {{ '{' }}
  const sum = numbers.reduce((acc, num) => acc + num, 0);
  return sum / numbers.length;
{{ '}' }}

3. Itera e raffina

Aggiungi più commenti o modifica il codice per ottenere risultati migliori.

// Crea una funzione che calcola la media di un array di numeri
// Gestisci il caso di array vuoto
function calculateAverage(numbers) {{ '{' }}
  if (numbers.length === 0) {{ '{' }}
    return 0;
  {{ '}' }}
  const sum = numbers.reduce((acc, num) => acc + num, 0);
  return sum / numbers.length;
{{ '}' }}

Consigli e Trucchi per GitHub Copilot

Sii specifico nei commenti

Più dettagliati sono i tuoi commenti, migliori saranno i suggerimenti di Copilot. Includi dettagli come tipi di dati, comportamenti attesi e casi limite.

Usa i docstring e i JSDoc

Formattare i commenti come docstring o JSDoc può aiutare Copilot a generare codice migliore con tipi corretti.

Inserisci firme di funzioni

Scrivere solo la firma della funzione (nome e parametri) può essere sufficiente per far generare a Copilot un'implementazione.

Usa Copilot per i test

Copilot eccelle nel generare casi di test per il tuo codice. Basta chiedere di creare test per una determinata funzione.

Rifiuta e riprova

Se il suggerimento non è quello che desideri, rifiutalo e riprova. Copilot mostrerà soluzioni alternative.
